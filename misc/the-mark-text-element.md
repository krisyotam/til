date: "2025-04-05"
tags: []
---

The [\<mark\>](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/mark) HTML element represents marked or highlighted text.

```html
<span><mark>Peter</mark> Mekhaeil</span>
```

Can be styled:

```css
mark {
  background-color: yellow;
  color: black;
}
```
